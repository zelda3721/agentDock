色彩 排版 间距·动效 溯源条 组件 界面 交付

鸿蒙三端 · 手机 / 平板 / 2in1

本地优先的智能体工作台：模型自选、数据在端、记忆可审计。

这不是一个聊天 App 的外观。本地 CPU 正在跑模型——tok/s、首 token 时延、内存预算、召回率、上下文水位是这个产品的材质，界面的职责是让它们在每一屏都可见、可查、可撤销。

立场 · 语义色轴

青 = 本地  /  琥珀 = 远程

贯穿模型芯片、消息色带、当前大脑、隐私围栏、档位表、知识库隐私级别。四重冗余强制：色相 + 填充实/空 + 图标形状 + 文字标签——只靠色相即缺陷。

立场 · 签名元素

溯源条（Provenance Rail）

每条助手消息下沿的一行仪表带：来源模型 · 引用 · 记忆 · 围栏拦截 · 耗时。它是"数据在端、可审计"的物质化，lg 的第三栏就是它的展开态。

立场 · 仪表排版

第二角色：等宽仪表体

凡系统测量的数——14.2 tok/s、412ms、0.87、ctx 62%——一律等宽 + tabular-nums。模型正文里的数字不做处理：仪表体的边界是"系统测量的数"。
